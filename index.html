Features

Fun Commands

Ask the bot "tell me a joke" to hear an offensive joke or maybe a normal one.

Moderation Tools

Manage your server with powerful moderation commands like ban, kick, mute, and more.

Hybrid Commands

Supports both traditional (?) and slash (/) command formats for maximum compatibility.

Commands

?clear or /clear

Clears a number of messages as the user specifies.

?ban or /ban

Bans a user from the server.

?warn or /warn

Warns a user for breaking rules.

?mute or /mute

Mutes a user for a specific duration.

?unmute or /unmute

Unmutes a previously muted user.

?kick or /kick

Kicks a user from the server.

?ping or /ping

Checks the bot's latency.
